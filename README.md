# Face Recognition Attendance System

## 📌 Description
This project is an automated attendance system that uses face recognition technology to detect and recognize individuals in real time. It replaces traditional attendance methods such as manual registers or fingerprint systems.

## 🚀 Features
- Real-time face detection using OpenCV
- Face recognition and identification
- Automatic attendance marking
- Efficient and contactless system

## 🛠 Tech Stack
- Python
- OpenCV
- NumPy

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies: pip install -r requirements.txt  
3. Run the program: python main.py
